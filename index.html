Project Overview

Developed as part of a collaborative GIS development project, this application uses Google Maps API, SQL databases, ArcGIS workflows, and iNaturalist observations to visualize pollinator distributions and invasive species threats across London, Ontario.

  • Integrated Google Maps API for interactive mapping.
  • Managed ecological datasets using SQL and GIS workflows.
  • Built species filters, district summaries, and density visualizations.
  • Supported biodiversity analysis through hotspot and overlay techniques.
